# 🎵🤖 Hand-Gesture AI Music Synthesizer

An interactive, AI-powered musical instrument that transforms hand gestures into sound in real time. By combining computer vision and audio processing, this project allows users to "play" music simply by showing their fingers to a webcam.

---

## 🌟 Features

- **AI-Powered Detection**  
  Utilizes MediaPipe Hand Landmarker for accurate and low-latency hand tracking.

- **Sample-Based Synthesis**  
  Plays high-quality `.wav` audio samples based on detected finger count (1–10 fingers).

- **Real-Time Interaction**  
  Designed for live performance with smooth transitions using cooldown and audio fade-out.

- **Dual Hand Support**  
  Detects both hands simultaneously, enabling a full 10-note range.
